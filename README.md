# simple-page

npm-scriptsを使った静的ページ(HTML5/CSS3)のミニマム開発環境


## 環境構築
* npm/yarn
  * browser-sync
  * node-sass
  * nodemon
  * rimraf


## 開発手順
```
yarn install
```
```
yarn run build:scss:watch
```
```
yarn run serve
```
```
open http://localhost:3000
```
